I'm currently working in Inlogic on tester position. All I do is I test games. I was really piss off of that. It's very boring work to do for programmer so I decided I will create tool which automatize testing at least little bit.


Core of this app use ADB(android debug tools). It basically send touches to phone screen randomly for time user choose. In this time it is bit slow because it take ADB some time to send touch. I was thinking to use ADB on phone but in order to do that you must have rooted phone. This is wall I currently facing so if you have any proposition how to make it working please let me know.
